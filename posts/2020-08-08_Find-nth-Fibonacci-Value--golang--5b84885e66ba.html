Find nth Fibonacci Value [golang]

Nth Fibonacci value can be find using recursion as well as iteration.

Normal recursion process includes complexity of:
Time Complexity : O(2^n)
Space Complexity : O(n)

Recursive approach with Dynamic Programming

Best way to find using recursion is by using memoization (Dynamic Programming) technique.
Time Complexity : O(n)
Space Complexity : O(n)

package main
import (
 "fmt"
)
func fib(n int, mem map[int]int) int {
 if n <= 1 {
  return n
 }
 if n == 2 {
  return 1
 }
 if val,ok := mem[n]; ok {
  return val
 }
 val := fib(n-1, mem) + fib(n-2, mem)
 mem[n] = val
 return val
}
func main() {
 mem := make(map[int]int)
 n := 6
 fmt.Printf("Fib value of %d : %d\n", n, fib(n, mem))
}

Iterative approach

But iteration uses the same time complexity but with O(1) space complexity.

Time Complexity : O(n)
Space Complexity : O(1)

package main
import "fmt"
func main(){
  num := 8
  res := fib(num)
  fmt.Printf("Fibanocci value of : %d is : %d\n",num, res)
}
func fib(n int) int {
  first := 0 ; second := 1
  if n == 0 {
    return first
  } else if n <= 2 {
    return second
  }
  for i := 2 ; i <= n; i++ {
    second = second + first
    first = second - first
  }
  return second
}

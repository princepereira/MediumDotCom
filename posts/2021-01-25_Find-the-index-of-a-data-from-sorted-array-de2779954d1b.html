Find the index of an element from sorted array

This clearly explains how we can find the index of a data from a sorted array in most fastest way O(log n).

Eg: Take a sample array : [-2, 3, 5, 8, 10, 13, 17, 19]

  1. Define left boundary index, right boundary index and middle index to narrow down the search. 
    left = 0
    right = len(arr) -1 = 7,
    middle = (left+right)/2 = 3
  2. Check the value = array[middle]
    If yes return the index
  3. Check the value < array[middle]
    If yes, then we have to narrow down the search between left and middle.
    So right = (middle-1) and middle = (left+right)/2
  4. If the value >array[middle]
    Then we have to narrow down the search between middle and right.
    So left = (middle+1) and middle = (left+right)/2
  5. Continue the process till you find the element or the condition (left<middle<right) is broken.
    If the element is found, return index, if not found return -1 because -1 can never be a valid index of an array.

Program

import java.util.*;
public class FindIdx{
  public static void main(String[] args){
    int[] arr = {-2, 3, 5, 8, 10, 13, 17, 19};
    FindIdx o = new FindIdx();
    System.out.println("Array : "+Arrays.toString(arr));
    System.out.println("Index of 10 : "+o.findIdx(arr, 10));
    System.out.println("Index of 19 : "+o.findIdx(arr, 19));
    System.out.println("Index of 14 : "+o.findIdx(arr, 14));
  }
  public int findIdx(int[] arr, int val){
    int l = 0;
    int r = arr.length -1;
    int m = (l+r)/2;
    return findIdx(arr, val, l, m, r);
  }
  private int findIdx(int[] arr, int val, int l, int m, int r){
    if (l > m || m > r) {
      return -1;
    }
    if (arr[m] == val){
      return m;
    }
    if (arr[m] > val) {
      r = m-1;
    } else {
      l = m+1;
    }
    m = (l+r)/2;
    return findIdx(arr, val, l, m, r);
  }
}

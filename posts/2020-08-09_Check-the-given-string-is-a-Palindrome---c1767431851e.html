Check the given string is a Palindrome ?

Use iteration technique from 0 → n/2 element and (n-1)th element to n/2th element same time and compare on each index.

for i = 0 to i = n/2:
    if ith value not equals (lastIdx - i)th value:
         return False
return True
package main
import "fmt"
func main() {
 str := "abcdcba"
 fmt.Printf("String : '%s' palindrome status : %t\n", str, isPalindrome(str))
}
func isPalindrome(str string) bool {
 lastIdx := len(str) - 1
 for i := 0; i < lastIdx/2 && i < (lastIdx-i); i++ {
  if str[i] != str[lastIdx-i] {
   return false
  }
 }
 return true
}

Time Complexity : O(n/2) = O(n)
Space Complexity : O(1)

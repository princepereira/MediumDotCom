Find Smallest Difference of 2 Numbers from 2 Arrays

This is an interview problem which is frequently asked in many interview questions.
You will be provided with 2 unsorted arrays and requirement is to find the smallest possible difference between the element from first array and element from second array.

Steps:

  1. Arrays are :

2. Sort both the arrays

3. Keep an index pointers on both the arrays starting from ‘index = 0’.
Maintain 3 variables ‘elem1’, ‘elem2’ and ‘diff’.
In the beginning elem1 = arr1[0] , elem2 = arr2[0] and diff = abs(arr2[0] — arr1[0])

4. Rules to follow:
 4.1. If the diff is ‘0’ then return because 0 is will be the most smallest difference.
 4.2. If the newly calculated ‘diff’ is lesser than previous ‘diff’, replace the values of ‘elem1’, ‘elem2’ and ‘diff’ with current values.
 4.3. If the arr1[i] < arr2[j], do i++ else j++
 4.4. Return the final elem1, elem2, diff value.

Code :

File : SmallestDiff.java

import java.util.*;
import java.lang.Math;
public class SmallestDiff {
  public static void main(String[] args){
    int[] arr1 = {23, 45, 12, 3, 4, -8, -9};
    int[] arr2 = {13, 5, 7, 63, -4};
    int[] arr = smallestDiff(arr1, arr2);
    System.out.println(Arrays.toString(arr));
  }
  static int[] smallestDiff(int[] arr1, int[] arr2){
    Arrays.sort(arr1);
    Arrays.sort(arr2);
    int[] result = new int[3];
    int elem1 = arr1[0], elem2 = arr2[0], diff = Math.abs(elem2 - elem1);
    int i = 0, j = 0;
    while(i < arr1.length && j < arr2.length){
      int tmpDiff = Math.abs(arr1[i] - arr2[j]);
      if (tmpDiff == 0) {
        result[0] = arr1[i]; result[1] = arr2[j]; result[2] = 0;
        return result;
      } else if (tmpDiff < diff){
        elem1 = arr1[i]; elem2 = arr2[j]; diff = tmpDiff;
      }
      if (arr1[i] < arr2[j]) {
        i++;
      } else {
        j++;
      }
    }
    result[0] = elem1; result[1] = elem2; result[2] = diff;
    return result;
  }
}

Output:

[4, 5, 1]

Time Complexity : O(nlogn + mlogm) 
[where ’n’ is the length of first array and ‘m’ is the length of second array. And ‘nlogn’ is the time taking for sorting arr1 and ‘mlogm’ is the time taking for sorting arr2.]
Space Complexity : O(1)

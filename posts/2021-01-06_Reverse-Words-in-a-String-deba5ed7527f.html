Reverse Words in a String

Given an input string s, reverse the order of the words.

Reverse Words in aÂ String

Given an input string s, reverse the order of the words.

A word is defined as a sequence of non-space characters. The words in s will be separated by at least one space.

Return a string of the words in reverse order concatenated by a single space.

Note that s may contain leading or trailing spaces or multiple spaces between two words. The returned string should only have a single space separating the words. Do not include any extra spaces.

Eg:1

Input: s = " the sky is blue "
Output: "blue is sky the"

Program

import java.util.*;
public class ReverseWord {
public static void main(String[] args){
    String str = "  World is beautiful  ";
    System.out.println("Before Reverse : '" + str + "'");
    char[] a = str.toCharArray();
    reverse(a, 0, a.length -1);
    reverseWords(a, a.length -1);
    str = cleanSpaces(a);
    System.out.println("After Reverse  : '" + str + "'");
  }
  // reverse the entire letters of a sentence
  // This will make "hello world" to "dlrow olleh"
  static void reverse(char[] a, int i, int j){
    while(i < j){
      char t = a[i];
      a[i++] = a[j];
      a[j--] = t;
    }
  }
  // reverse the letters of reversed words
  // This makes "dlrow olleh" to "world hello"
  static void reverseWords(char[] a, int n){
    int i = 0, j = 0;
    while(i < n && j < n){
      while(i < n && a[i] == ' ') i++;
      j = i;
      while(j <= n && a[j] != ' ') j++;
      reverse(a, i, j-1);
      i = j +1;
    }
  }
  // This removes the leading and trailing spaces by
  // - finding i where the real letter starts (earlier parts will be spaces)
  // - finding j where the real letter ends (rest part will be spaces)
  // - make a string from the array starting with i and ending with j
  static String cleanSpaces(char[] a) {
    int i = 0;
    int j = a.length - 1;
    while(i < j){
      if (a[i] == ' ') i++;
      if (a[j] == ' ') j--;
      if (a[i] != ' ' && a[j] != ' ') break;
    }
    if (i >= j){
      return "";
    }
    StringBuffer sb = new StringBuffer();
    while(i <= j){
      sb.append(a[i++]);
    }
    return sb.toString();
  }
}

Output

Before Reverse : '  World is beautiful  '
After Reverse  : 'beautiful is World'

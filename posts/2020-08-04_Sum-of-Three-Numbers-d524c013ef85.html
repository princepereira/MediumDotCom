Sum of Three Numbers

How to find triplets with sum ‘0’ in an array ?

  1. Given array

2. Sort the array

3. Choose a current node, left pointer and a right pointer. Current node is first element of the sorted array, left pointer is the left most index after current node and right pointer is the right most element of the array.

4. Find the sum of 3 node values :

Formula is :

sum = arr[curNode] + arr[lPtr] + arr[rPtr]

4.1. if the sum == 0 → consider the value for the triplet and proceed for the next triplet by incrementing lPtr and decrementing right pointer.

[lPtr++, rPtr — ]

4.2. if the sum < 0 → increment the lPtr [lPtr++]

4.3. if the sum > 0 → decrement the rPtr [rPtr — ]

Program:

File: ThreeNumberSum.java

import java.util.*;
public class ThreeNumberSum {
   public static void main(String[] args){
       int[] arr = {12, 1, 3, 2, -6, 5, -8, 6};
       Arrays.sort(arr);
       System.out.println(Arrays.toString(arr));
       ThreeNumberSum obj = new ThreeNumberSum();
       ArrayList<int[]> triplets = obj.getTripletsForSum0(arr);
       for(int[] val : triplets){
          System.out.println(Arrays.toString(val));
       }
  }
  ArrayList<int[]> getTripletsForSum0(int[] arr){
      ArrayList<int[]> arrList = new ArrayList<>();
      int lPtr = 0, rPtr = 0, sum = 0;
      for(int i = 0; i < arr.length - 3; i++){
         for(lPtr = i + 1, rPtr = arr.length - 1; lPtr < rPtr; ){
            sum = arr[i] + arr[lPtr] + arr[rPtr];
            if(sum == 0){
               int[] triplet = {arr[i], arr[lPtr], arr[rPtr]};
               arrList.add(triplet);
               lPtr++; rPtr--;
            } else if(sum < 0){
               lPtr++;
            } else {
               rPtr--;
            }
         }
      }
      return arrList;
   }
}

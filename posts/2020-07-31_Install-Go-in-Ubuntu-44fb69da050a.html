Install Go in Ubuntu

Here I will explain how to install Go [Version: 1.14] in Ubuntu [Ubuntu 16.04] and run your first Go Project.

Download Go binary tarball to your Ubuntu machine [server/vm].

$ wget https://golang.org/dl/go1.14.6.linux-amd64.tar.gz

Untar the binary tarball to ‘/usr/local’ directory.

$ sudo tar -C /usr/local -xzf go1.14.6.linux-amd64.tar.gz

Set your Bin Path and GOPATH in .bashrc file and source the file.

$ mkdir -p $HOME/GoProjects/src -p $HOME/GoProjects/pkg -p $HOME/GoProjects/bin
$ echo "export PATH=$PATH:/usr/local/go/bin:$HOME/GoProjects/bin" >> $HOME/.bashrc
$ echo "export GOPATH=$HOME/GoProjects" >> $HOME/.bashrc && source $HOME/.bashrc

Your Go setup is ready.

Create your first Go Project.

$ cd $HOME/src/GoProjects
$ mkdir SampleProject
$ cd SampleProject
$ vi main.go
//================Code starts here ==============//
package main
import "fmt"
func main(){
    fmt.Println("Hello World...")
}
//================Code ends here ==============//

Run your Go Project.

$ go run main.go
Output :
Hello World...
Or using
$ go install .
$ cd ../bin
$ ./Sample
Output :
Hello World...
